# Conversor PDF para Imagem

Converta um arquivo PDF inteiro em uma única imagem PNG.
Ideal para quem busca juntar arquivos que foram separados por páginas

## Bibliotecas

- [Lucide Icons](https://lucide.dev/icons/)
- [Next.js](https://nextjs.org/)
- [PDF.js (dist)](https://www.npmjs.com/package/pdfjs-dist)
- [React](https://react.dev/)
- [Serwist](https://serwist.pages.dev/)
- [Tailwind CSS](https://tailwindcss.com/)

## Início Rápido

```bash
# Instalar depenências
yarn

# Rodar o projeto
yarn dev
```
